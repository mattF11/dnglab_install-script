# dnglab_install-script
A script to install and peform basic operations with dnglab.
As i'm learning bash scripting for a simple and amateur use,these scripts can be usefull,in this case,installing and perform basic operations with dnglab.  
The idea of the bash script came as i felt the need to semplify my workflow: easily  convert my .ARW files without wasting time,the need to lookout for notes,documentation and the desire to simply run and select the settings i need.

## Dependancies
1. bash shell
2. cargo
3. rust programming language

### Documentation
Introductive documentation is provided both in italian and english(.org,pdf),only the essential parts are explained as it's really easy to use and you only need to follow the instructions.

#### Features
1. dnglab install
2. dng conversion(single image or folder)
